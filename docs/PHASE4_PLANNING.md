### 8. 数据库读写分离

**架构**:
```
应用层
  │
  ├─ 写操作 ──▶ 主库 (PostgreSQL Primary)
  │              │
  │              ▼
  │          从库 1 (Replica)
  │              │
  │              ▼
  └─ 读操作 ──▶ 从库 2 (Replica)
```

**实现**:
```typescript
// Prisma 数据源配置
datasource primary {
  provider = "postgresql"
  url      = env("DATABASE_PRIMARY_URL")
}

datasource replica {
  provider = "postgresql"
  url      = env("DATABASE_REPLICA_URL")
}

// 读写路由
const db = query.startsWith('SELECT') 
  ? prisma.$queryRaw(query)  // 读从库
  : prisma.$executeRaw(query); // 写主库
```

### 9. 消息队列

**架构**:
```
应用层
  │
  ▼
RabbitMQ
  │
  ├─ 邮件队列 ──▶ 邮件服务
  ├─ 通知队列 ──▶ 通知服务
  ├─ 任务队列 ──▶ 任务执行服务
  └─ 日志队列 ──▶ 日志服务
```

**实现**:
```typescript
// 生产者
await channel.sendToQueue('email_queue', 
  Buffer.from(JSON.stringify({ to, subject, body }))
);

// 消费者
channel.consume('email_queue', async (msg) => {
  const { to, subject, body } = JSON.parse(msg.content.toString());
  await sendEmail(to, subject, body);
  channel.ack(msg);
});
```
