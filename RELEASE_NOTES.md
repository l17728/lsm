# LSM v3.2.2 正式发布说明

**版本**: 3.2.2 | **发布日期**: 2026-03-28 | **代号**: Calendar Enhancement

---

## 🚀 版本概述

LSM v3.2.2 重点增强了预约系统的日历体验，完善了审批权限体系，并大幅更新了用户文档。

---

## ✨ 新功能亮点

### 1. 日历系统全面升级

**月历预约可视化**:
- 预约项显示清晰图标和 Badge 数量标识
- 有预约的日期蓝色边框高亮
- 状态颜色边框区分不同状态
- 悬停显示预约详情 Tooltip

**日历导航优化**:
- 点击日期单元格 → 进入日视图
- 点击预约项 → 进入日视图 + 高亮该预约
- 月/周/日视图无缝切换

### 2. 审批权限体系

| 角色 | 服务器预约审批 | 集群预约审批 |
|------|----------------|----------------|
| ADMIN | ✅ | ❌ |
| SUPER_ADMIN | ✅ | ✅ |

### 3. AI 时间推荐 🆕

集群预约支持 AI 智能推荐最佳预约时间段：
- 基于历史使用模式分析
- 多维度评分：时间邻近度、避开高峰、冲突检测
- 返回 Top 5 推荐时间段

## 📦 升级指南

### 从 v3.2.1 升级

```bash
# 1. 备份数据
pg_dump lsm > lsm_backup_$(date +%Y%m%d).sql

# 2. 拉取代码
git checkout v3.2.2

# 3. 更新依赖
cd src/backend && npm install && cd ../frontend && npm install

# 4. 数据库迁移（如有）
cd src/backend && npx prisma migrate deploy

# 5. 重启服务
docker-compose down && docker-compose up -d
```
