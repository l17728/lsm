## 一、项目概述

### 1.1 项目背景

LSM 项目是为实验室服务器集群设计的一体化管理平台，旨在解决传统服务器管理效率低、资源分配不透明、故障响应滞后等痛点问题。

### 1.2 项目目标

- 实现服务器全生命周期管理（申请、分配、监控、回收）
- 提供 GPU 资源智能调度与预测能力
- 构建高可用、高性能、安全可靠的运维平台
- 支持多终端访问（Web、移动端）

### 1.3 项目范围

| 模块 | 范围说明 |
|------|---------|
| 用户认证 | JWT + 2FA + RBAC 权限控制 |
| 服务器管理 | CRUD、状态监控、批量操作 |
| GPU 管理 | 分配、预测、负载均衡 |
| 任务调度 | AI 智能调度器、任务队列 |
| 监控告警 | Prometheus + Grafana、告警降噪 |
| 自动化运维 | 自动扩缩容、故障自愈 |
