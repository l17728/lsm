## 📊 功能概述

- 自动降级模拟数据

### 3. 成本核算可视化代码

**实现内容**:
- 成本分布饼图（GPU计算、CPU资源、内存、网络、存储）
- 按类别成本条形图
- 成本趋势面积图
- 服务器成本效率对比图

## 📈 数据结构

### AnalyticsSummary
```typescript
{
  totalCost: number;          // 总成本
  costTrend: number;          // 成本趋势
  avgUtilization: number;     // 平均利用率
  utilizationTrend: number;   // 利用率趋势
  peakResource: {             // 峰值资源
    type: string;
    value: number;
    time: string;
  };
  efficiency: number;         // 效率评分
  savings: number;            // 预估节省
}
```

### CostBreakdownItem
```typescript
{
  category: string;    // 类别名称
  amount: number;      // 金额
  percentage: number;  // 占比
  trend: number;       // 趋势
}
```

### ServerUtilization
```typescript
{
  serverId: string;
  serverName: string;
  cpuCores: number;
  totalMemory: number;
  cpuUsage: number;
  memoryUsage: number;
  gpuCount: number;
  gpuUsage: number | null;
  utilization: number;
  cost: number;
  efficiency: number;
}
```

## 📊 图表类型

1. **ComposedChart** - 资源使用趋势（面积图+折线图组合）
2. **PieChart** - 成本分布饼图
3. **BarChart** - 按类别成本、效率评分
4. **AreaChart** - 成本趋势
5. **热力图卡片** - 服务器利用率分布

## 🚀 使用方式

1. 登录系统后，点击侧边栏 "Analytics" 菜单
2. 选择时间范围（默认7天）
3. 切换标签页查看不同分析维度
4. 点击 "Export Report" 导出报告

## 📝 注意事项

1. 当数据库无历史数据时，系统会生成模拟数据用于展示
2. 成本费率可在 `analytics.service.ts` 中配置
3. 服务器规格从 `metadata` 字段读取，默认值：32核/128GB内存
4. 支持 WebSocket 实时更新
